# Applitechture Website

Marketing site for Applitechture — applitechture.com.

## Develop

    npm install
    npm run dev

## Environment

| Variable         | Purpose                                                                                       |
| ---------------- | --------------------------------------------------------------------------------------------- |
| `RESEND_API_KEY` | Resend API key for the contact form. Set in Vercel project settings; locally in `.env.local`. |
